1. Deploy Zabbix docker image, by running command "docker run -d -p 10051:10051 -p 10052:10052 -p 9001:80 -p 2812:2812 --name zabbix  berngp/docker-zabbix"

2. Get local 80 port mapping by running "docker ps -f name=zabbix"  Find the port that is mapped to 80 for this container.

3. Open VirtualBox and open the settings for the boot2docker-vm. On the sidebar, click 'Network', 
   and then click on 'Port Forwarding'. Create a new port and set both 'Host Port' and 'Guest Port'
   to the port mapped to 9001. The rest of the fields can be left to there default values.
   
4. Go to the Zabbix server (http://localhost:9001/zabbix). Login as username: Admin and password: zabbix.

5. Go to Configuration -> Hosts -> click on Zabbix Server -> Web scenarios.

6. At the "Configuration of Web Monitoring" page, click the "Create Scenario" button on the top right corner.

7. Enter scenario name, "MedExplorer", and select application, "General". Change update interval, if necessary.

8. Click the "Add" button at the bottom of the page.

9. Click the "Steps" tab.

10. Inside the steps box, click the "Add" link.

11. Enter the necessary information for the URL to listen to and specify the url to listen to: (http://10.2.101.113:9000/). Click the "Add" button.
